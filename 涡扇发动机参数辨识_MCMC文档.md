# 涡扇发动机效能参数贝叶斯辨识报告

## 一、问题描述

### 1.1 工程背景

涡扇发动机的性能由一组内部热力学参数共同决定，这些参数（如各部件效率、总压恢复系数）在实际工程中难以直接测量，只能通过外部可观测的整机性能指标（单位比推力 $R_{ud}$ 和单位比油耗 $C_{ud}$）进行间接辨识。由于测量噪声的存在，参数辨识问题具有不确定性，因此需要借助贝叶斯推断框架对参数的后验分布进行定量估计。

### 1.2 待辨识参数

共涉及 13 个参数，均为无量纲效率或总压恢复系数，其先验范围（均匀分布上下界）如下表所示：

| 编号 | 参数符号 | 物理含义 | 下界 $l_b$ | 上界 $u_b$ | 真值 $\theta^*$ |
|------|----------|----------|-----------|-----------|----------------|
| 1 | $\eta_k$ | 压气机绝热效率 | 0.84 | 0.86 | 0.8500 |
| 2 | $\eta_t$ | 涡轮绝热效率 | 0.86 | 0.92 | 0.8900 |
| 3 | $\eta_m$ | 机械效率 | 0.980 | 0.995 | 0.9880 |
| 4 | $\eta_v$ | 风扇效率 | 0.85 | 0.87 | 0.8600 |
| 5 | $\eta_{tv}$ | 风扇涡轮效率 | 0.90 | 0.92 | 0.9100 |

| 7 | $\eta_{c2}$ | 二次喷管效率 | 0.92 | 0.94 | 0.9300 |

## 二、前向模型

前向模型 `engine_forward(θ, cond)` 描述从参数向量 $\theta$ 到可观测性能指标的映射：

$$\mathbf{y} = \mathcal{F}(\theta) = [R_{ud},\ C_{ud}]^\top$$

### 2.1 主要计算步骤

**步骤 1：飞行速度与进口驻点温度**

$$a = \sqrt{k_{air} R_{air} T_H}, \quad V_\infty = a \cdot M_\infty$$

$$\tau_v = \left(1 + \frac{V_\infty^2}{2 \frac{k_{air}}{k_{air}-1} R_{air} T_H}\right)^{\frac{k_{air}}{k_{air}-1}}, \quad T_B = T_H \tau_v$$

**步骤 2：压气机出口温度**

$$T_k = T_B \left(1 + \frac{\pi_k^{(k_{air}-1)/k_{air}} - 1}{\eta_k}\right)$$

**步骤 3：相对耗油量**

$$g_T = 3\times10^{-5} T_g - 2.69\times10^{-5} T_k - 0.003$$

**步骤 4：热恢复系数**

$$\lambda_{heat} = \frac{1 - \dfrac{W_c}{h_g / \eta_k}}{1 - \dfrac{W_c}{h_g \eta_k \eta_t}}$$

其中 $W_c = \frac{k_{air}}{k_{air}-1} R_{air} T_B (\pi_k^{(k_{air}-1)/k_{air}} - 1)$，$h_g = \frac{k_T}{k_T-1} R_T T_g$。

**步骤 5：单位自由能**

$$L_{sv} = \lambda_{heat} \left[\frac{k_T}{k_T-1} R_T T_g \left(1 - \left(\tau_v \sigma_{bx} \pi_k \sigma_{kask} \sigma_{ks}\right)^{-\frac{k_T-1}{k_T}}\right) - \frac{W_c}{(1+g_T)\eta_k \eta_T \eta_t \eta_m (1-\delta)}\right]$$

其中 $\sigma_{bx} = \sigma_{cc} \cdot \sigma_{kan}$，$\delta$ 为涡轮冷却引气系数（随 $T_g$ 分段线性）。

**步骤 6：最优能量分配系数**

$$x_{pc} = \frac{1 + \dfrac{m V_\infty^2}{2 L_{sv} \eta_{tv} \eta_v \eta_{c2}}}{1 + \dfrac{m \eta_{tv} \eta_v \eta_{c2}}{\eta_{c1} \lambda}}$$

**步骤 7：单位比推力与比油耗**

$$V_{j1} = (1+g_T)\sqrt{2\eta_{c1}\lambda x_{pc} L_{sv}} - V_\infty$$

$$V_{j2} = \sqrt{2\frac{1-x_{pc}}{m} L_{sv} \eta_{tv} \eta_v \eta_{c2} + V_\infty^2} - V_\infty$$

$$R_{ud} = \frac{V_{j1}}{1+m} + \frac{m V_{j2}}{1+m}, \quad C_{ud} = \frac{3600\, g_T (1-\delta)}{R_{ud}(1+m)}$$

### 2.2 分段物性参数

燃气绝热指数 $k_T$ 和气体常数 $R_T$ 随温度按以下规则取值：

| $T_g$ 范围 | $k_T$ | $R_T$ (J/(kg·K)) |
|-----------|-------|-----------------|
| $\leq 1400$ K | 1.33 | 287.6 |
| 1400–1600 K | 1.30 | 288.0 |
| $> 1600$ K | 1.25 | 288.6 |

冷却引气系数：$\delta = \text{clip}\left(0.02 + (T_g - 1200)/100 \times 0.02,\ 0,\ 0.15\right)$

## 三、算法设置

### 3.1 贝叶斯框架

#### 先验分布

对参数向量 $\theta$ 采用均匀先验：

$$p(\theta) = \begin{cases} 1 & \theta \in [l_b,\, u_b] \\ 0 & \text{otherwise} \end{cases}$$

等价的对数先验：
$$\ln p(\theta) = \begin{cases} 0 & \theta \in [l_b,\, u_b] \\ -\infty & \text{otherwise} \end{cases}$$

#### 似然函数

观测模型假设加性高斯噪声：

$$R_{obs} = R_{ud}(\theta) + \varepsilon_R, \quad \varepsilon_R \sim \mathcal{N}(0, \sigma_R^2)$$
$$C_{obs} = C_{ud}(\theta) + \varepsilon_C, \quad \varepsilon_C \sim \mathcal{N}(0, \sigma_C^2)$$

噪声标准差取真值的 1%：$\sigma_R = 0.01 |R_{true}|$，$\sigma_C = 0.01 |C_{true}|$。

对数似然：

$$\ln p(\mathbf{y}_{obs} \mid \theta) = -\frac{1}{2}\left[\left(\frac{R_{obs}-R_{ud}}{\sigma_R}\right)^2 + \ln(2\pi\sigma_R^2) + \left(\frac{C_{obs}-C_{ud}}{\sigma_C}\right)^2 + \ln(2\pi\sigma_C^2)\right]$$

#### 后验分布

$$\ln p(\theta \mid \mathbf{y}_{obs}) = \ln p(\theta) + \ln p(\mathbf{y}_{obs} \mid \theta)$$

### 3.2 MCMC 算法：自适应 Metropolis-Hastings

#### 参数空间变换

为天然满足参数有界约束，将原始参数映射到单位超立方体：

$$z_i = \frac{\theta_i - l_{b,i}}{u_{b,i} - l_{b,i}} \in [0,1]$$

在 $z$ 空间上执行随机游走，越界时采用**反射边界**处理：

```
若 z < 0 → z = -z
若 z > 1 → z = 2 - z
```

反射后再映射回原始参数空间：$\theta = l_b + z \cdot (u_b - l_b)$。

#### 提议分布

$$z^* = z^{(t)} + \varepsilon, \quad \varepsilon \sim \mathcal{N}(\mathbf{0},\, \sigma_{prop}^2 \mathbf{I})$$

各维度共享同一提议标准差 $\sigma_{prop}$。

#### 接受准则（对数域）

$$\alpha = \min\left(1,\ \frac{p(\theta^* \mid \mathbf{y})}{p(\theta^{(t)} \mid \mathbf{y})}\right)$$

等价地，当 $\ln u < \ln p(\theta^*|\mathbf{y}) - \ln p(\theta^{(t)}|\mathbf{y})$（$u \sim U[0,1]$）时接受。

#### 自适应步长调整

| 配置项 | 取值 |
|--------|------|
| 初始提议标准差 $\sigma_{prop}$ | 0.012 |
| 自适应区间 | 步数 500–2500 |
| 调整间隔 | 每 100 步 |
| 目标接受率区间 | [0.18, 0.35] |
| 步长调整因子 | 偏低 → ×0.9；偏高 → ×1.1 |
| $\sigma_{prop}$ 限幅 | $[10^{-4},\ 0.5]$ |

自适应仅在烧入期内进行，后验采样阶段步长固定，以保证链的平稳性。

#### 采样配置汇总

| 配置项 | 取值 |
|--------|------|
| 总采样步数 | 10,000 |
| 烧入期（Burn-in）| 2,000 步 |
| 后验有效链长度 | 8,000 步 |
| 随机种子 | 42（主程序）/ 123（数据生成）|
| 初始点 $\theta_0$ | 各参数先验区间中点 |

### 3.3 后验统计量

从后验链 `chain_post`（去除烧入期后的 8000 步）提取：

- **后验均值**：$\bar\theta = \dfrac{1}{N}\sum_{s=1}^N \theta^{(s)}$
- **MAP 估计**：后验链中对数后验最大处的样本：$\hat\theta_{MAP} = \arg\max_s \ln p(\theta^{(s)}|\mathbf{y})$
- **95% 可信区间**：$[\text{quantile}(2.5\%),\ \text{quantile}(97.5\%)]$

## 五、分析

### 5.1 算法收敛性分析

**链轨迹（Trace Plot）** 是判断 MCMC 收敛的主要可视化工具。收敛良好的链应表现为：
- 烧入期后链在参数真值附近平稳振荡，无明显趋势漂移；
- 链的探索范围覆盖后验分布的主要质量区域；
- 自相关长度（ACL）适中，不过长也不过短。

**接受率** 是评估提议分布质量的关键指标。理论上，对于高维问题，最优接受率约为 23%（无限维极限）；对于低维问题约为 44%。本代码目标接受率设为 [0.18, 0.35]，落在合理范围内。

### 5.2 可辨识性与参数敏感性

由于输出观测量只有两个（$R_{ud}$、$C_{ud}$），而待辨识参数多达 13 个，系统存在**欠定性**：

- **强可辨识参数**：对输出有显著单调影响的参数（如 $\eta_k$、$\eta_t$、$\eta_T$ 等），后验分布会较集中，95% 可信区间明显窄于先验区间。
- **弱可辨识参数**：对输出影响较小、或与其他参数存在相关性的参数（如部分总压恢复系数），后验分布可能接近均匀先验，95% 可信区间接近先验范围，说明数据对这些参数的约束力弱。
- **参数相关性**：后验分布中可能出现参数间的负相关或正相关，意味着观测数据只能约束参数的某种组合，而非各参数独立值。

### 5.3 噪声水平的影响

本实验设置噪声水平为 1%（$\sigma_R = 0.01 R^*$，$\sigma_C = 0.01 C^*$），这在工程测量中属于较低噪声水平。噪声越小，似然函数越尖锐，后验分布越集中，辨识精度越高。若噪声水平增大（如 5%–10%），则：

- 后验分布变宽，95% 可信区间增大；
- 后验均值与 MAP 估计的误差增大；
- 弱可辨识参数的后验分布更接近先验。

### 5.4 先验均匀分布的合理性

均匀先验反映了工程专家知识：各参数取值被限制在合理的物理范围内（如效率不超过 1，总压恢复不低于 0.94 等），但在该范围内无偏好。这是一种保守而客观的假设，避免了对先验信息的过度依赖。若有更精确的先验知识（如高斯先验），可进一步收窄后验不确定性。

### 5.5 后验均值 vs. MAP 估计

- **后验均值** $\bar\theta$ 是 $L_2$ 损失下的贝叶斯最优估计，对后验分布的整体形状敏感，适合后验接近对称分布的情形。
- **MAP 估计** $\hat\theta_{MAP}$ 是 $0$-$1$ 损失下的最优估计，等价于后验的众数，对后验链样本中的偶然极值敏感，链长越短，MAP 越不稳定。

当后验分布接近单峰对称高斯时，两者趋于一致。差异较大时，通常说明后验分布存在明显偏斜或多峰结构。

### 5.6 反射边界的作用

参数的有界约束若用截断（Truncation）或拒绝方案处理，会导致边界处采样不足，使后验分布在边界附近被人为压低。本代码采用**反射边界**，相当于在 $[0,1]$ 范围内对 $z$ 空间作折叠反射，保证了边界附近的正确采样密度，是处理有界均匀先验的标准方法。

### 5.7 局限性与改进方向

| 局限性 | 潜在改进方向 |
|--------|-------------|
| 单链 MCMC，难以诊断收敛 | 多链并行运行，计算 Gelman-Rubin 统计量 $\hat{R}$ |
| 各维度共享步长，适应性有限 | 使用全协方差自适应 MCMC（AM-MCMC）|
| 仅有两个观测量，系统欠定 | 增加多工况（不同 $M_\infty$、$\pi_k$、$T_g$）数据 |
| MAP 由链内最优样本确定，精度受链长限制 | 后处理阶段用梯度优化精化 MAP |
| 链相关性未量化 | 计算有效样本量（ESS）与积分自相关时间（IACT）|

## 六、结论

本工作构建了涡扇发动机热力学参数的贝叶斯辨识框架，核心贡献包括：

1. **前向模型**：实现了从 13 个效率/总压恢复参数到单位比推力和比油耗的完整热力学计算链，包含分段物性参数和涡轮冷却引气效应；
2. **贝叶斯推断**：在均匀先验（参数物理范围约束）与高斯似然（1% 测量噪声）框架下，建立了参数的后验分布模型；
3. **自适应 MCMC**：采用带反射边界的自适应 Metropolis-Hastings 算法，在变换空间内高效采样 10,000 步（有效后验链 8,000 步），自适应步长确保接受率在合理范围内；
4. **不确定性量化**：提供后验均值、MAP 估计及 95% 可信区间，量化了单次测量条件下各参数的辨识不确定性。

该框架为航空发动机健康监测与性能退化诊断提供了方法论基础，可扩展至多工况数据融合与实时参数跟踪场景。
